Akebi 简易教学 #

Akebi 零基础使用教学。

INFO

此文章是由 @ΣI 与 @GALAGALA 共同编写的 Akebi 簡易說明書 v1.3 并经过 @AhAlpha 的简单修改。

目录 #

首要 - 在进入大门之前，这里是必须的。

购买 - 成为尊贵的 Akebi 用户的第一步。

  • 官方渠道 - 较为安全的渠道。

  • 非官方渠道 - 可能会有风险的渠道。

绑定 - 将你的密钥绑定至账号，以便于获取身份组和管理密钥。

  • 绑定至 Discord 的账号 - 使用机器人绑定密钥。

  • 获取身份组 - 获取内部频道查看权限。

  • 密钥管理界面 - 管理你的密钥。

    • 重置 HWID - 在更换电脑时，需要重置它。

    • 暫停时间 - 在不使用 Akebi 时，可以暂停它。

游玩 - 下载与运行 Akebi ，并与它一起游览提瓦特。

错误 - 一些无法正常游玩的问题解答。

0 首要 #

由于那个"好日子"越来越近了，你必须遵守这些规则:

01 在所以公共平台中，你不能宣传Akebi，包括其功能和更新消息。

02 在 Akebi Discord 服务器 中，你也不能在非 PRIVATE 类别频道讨论功能和更新消息，否则这将作为米哈游指证 Akebi 的证据。

03 在 Akebi Discord 服务器 中，你非常不能在非 #nsfw 频道发布一些R18的内容。

服务器内的违规消息一般由 @MDMA#4044 来处理。在CN频道中也可由 @Kans#2528 删除。

1 购买 #

目前有两种购买 Akebi 的渠道，分别是与。

官方渠道 #

较为安全的渠道。

  • 如果出现密钥问题，需要在 Discord 私聊 @Callow#1429 来解决。

  • 如果出现其他问题，需要自行解决，或者阅读下文的后，在 Akebi Discord 服务器寻求帮助。

前往 #

可在 Akebi Discord 服务器 上的 #how-to-buy-official 频道获取官方的最新购买地址。

  • Sellix - 仅支持加密货币。

非官方渠道 #

可能会有风险的渠道，也被称为代购。

  • 在不能通过购买时，可以从此渠道购买。

  • 如果出现问题，需要私聊 卖给你密钥的代购 来解决。

  • 不建议去购买日卡超过20元，周卡超过70元，月卡超过170元的代购。

01 Discord #

相比而言，比较放心的代购会在 Akebi Discord 服务器 服务器帮助你购买密钥。而在服务器里拥有黄色昵称的代购处购买，应该不会有太大的风险。

前往 #

可在 Akebi Discord 服务器 上的 #resellers-forum 频道选择适合于你的代购。

  • 小飞棍!ce丨CN区代购 支持微信 花呗 支付宝 银行卡 - 支持微信/支付宝/花呗。
  • GALAGALA |黑心代购| - 支持微信/支付宝/银行汇款/PayPal。
  • 往生堂堂主|代购买服务 - 支持微信/支付宝。
  • 荣耀丨CN 国人 港澳台同胞 代购互助贴 - 支持微信/支付宝/花呗/闲鱼/信用卡。
  • CN区 神里代购 - 支持微信/支付宝/闲鱼。
  • [🇨🇳|🇭🇰|🇲🇴]Drich 闲鱼|微信|支付宝 24h自动发货 - 支持微信/支付宝/闲鱼/境外汇款。
  • (...)

如果因为一些其他因素，也可以选择非黄色昵称的代购购买，但是不推荐。

02 闲鱼等其他渠道 #

相对来说风险较大，并且价格较高，不太推荐。

2 绑定 #

至此，你应该已经获取到了你的 Akebi 密钥。接下来，你需要将其绑定在自己的 Discord 账号上，以便后续的密钥暂停与重置并且获取服务器中的 Insider 身份组。

绑定至 Discord 的账号 #

这是关键。提示，在 Discord 绑定密钥并不会因此开始计时。

00 需要允许服务器成员直接向您发起私聊处于启用状态。

01 在 Akebi Discord 服务器 中，在 PRIVATE 类别找到并进入 #license-bot 频道，点击里面的 Ask for panel 按钮。

02 此时这里将会提示You will get DM message soon.并收到一条私聊消息，点击机器人头像以进入私聊界面。

03 依次点击 Licenses info -> Add 以进入密钥添加界面。

04 找到你的 密钥 ，并将其复制至 输入框 后，点击 提交 ，此时你已经完成了密钥添加工作。

获取身份组 #

在添加了密钥之后你可以获取 Insider 身份组，以查看内部频道来获取更新资讯和讨论功能。请注意，它只能持续到你的所有密钥结束。

00 如果你目前在 Licenses info 界面，你需要点击 Back 回到主界面。

01 点击 Get role 按钮即可获取身份组。

密钥管理界面 #

管理单个密钥。

01 依次点击 Licenses info -> Details 并输入你想要管理的密钥后点击 提交 ，以进入密钥管理界面。

重置 HWID #

如果需要更换电脑，则需要重置 HWID。请注意，只有周卡和月卡可以重置，并且分别只有1次和3次重置机会。

01 点击 Reset HWID 即可重置此卡密的 HWID。

暫停时间 #

如果你暂时一段时间不会使用 Akebi，你可以选择暂停密钥时间。请注意，只有1次暂停机会。

01 点击 Pause 即可暂停此卡密的时间。如需恢复，则需要点击 Unpause 。

3 游玩 #

你终于完成了上面的准备工作，那就开始游玩 Akebi 吧。

01 使用你购买后收到的 下载链接 下载 AkebiPrivate.zip 。

02 打开下载好了的 AkebiPrivate.zip 并将 AkebiLauncher.exe 解压至一个固定目录。

03 启动 AkebiLauncher.exe ，然后会出现一个应用选择弹窗，你需要选择原神主程序。

如果在此处出现错误，请查看。

国服默认路径: C:\Program Files\Genshin Impact\Genshin Impact Game\YuanShen.exe

国际服默认路径: C:\Program Files\Genshin Impact\Genshin Impact Game\GenshinImpact.exe
