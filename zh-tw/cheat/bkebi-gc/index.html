Bkebi-GC #

#

同Acrepi，属于Akebi-GC系的衍生端。

INFO

本文章截至最后一次更新此作弊端的内容的时间为2023-3-4 9:48 此时Bkebi-GC的最新版本为v1.1.1。

简单介绍 #

相比Acrepi，拥有更快速度的更新，并且修复了许多Bug和增加了许多新功能。不过自Bkebi-1.1.0开始，加入了一种验证机制，并且每天名额有限。

主要功能 #

在的基础上增加了: 元素队列模式 移动增强 不死 远程工具 自动解密 元素光环 使用AFK攻击 怪物替换 自定义传送 Pro 菜单解锁器 ...
