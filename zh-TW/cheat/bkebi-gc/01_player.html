玩家 #

與角色有關的各種屬性。

攻擊效果 #

調整角色攻擊的傷害效果。

啟用 #

是否啟用、、以及。

倍攻模式 #

使一次攻擊造成多次傷害。

  • 一共有三種模式: 、以及。

  • 需要上面的按鈕處於勾選狀態才會生效。

倍率 #

更改一次攻擊能造成幾次傷害。

  • 僅在和 關閉的情況下生效。

  • 使用此功能可能會面臨風險。

  • 範圍: 2 ~ 1000

秒殺模式 #

根據使怪物能夠倒下的血量 決定一次攻擊造成的幾次傷害。

  • 會覆蓋上面的。

  • 會覆蓋下面的。

  • 一次攻擊最大200倍率。

隨機倍攻 #

更改一次攻擊能造成隨機的幾次傷害。

  • 會覆蓋上面的。

  • 僅在關閉的情況下生效。

倍功最小值 #

調整隨機倍攻的最小值。

  • 範圍: 1 ~ 1000

倍功最大值 #

調整隨機倍攻的最大值。

  • 範圍: 2 ~ 1000

普攻範圍 #

使周圍的生物承受被攻擊生物受到的傷害。

  • 傷害僅會顯示在被攻擊生物的身上。

  • 類似於ID傷害。

範圍 (米) #

設定的生效範圍。

  • 範圍: 5 ~ 50

多重動畫 (多重動畫倍率)  #

使全局攻擊等動畫在攻擊的一瞬間重複播放 並能造成多次傷害。

  • 可以作用於生物。

  • 多人遊戲下僅限成為時對一定範圍內的生物有效。

  • 使用此功能可能會面臨風險。

  • 範圍: 1 ~ 150

使用舊的重複模式 (動畫狀態) #

使在攻擊的過程中重複 而不是一瞬間。

  • 範圍: 0 ~ 2

僅玩家 #

使僅對當前上場角色生效 並且不會生效於實體。

攻擊速度 #

角色在攻擊時加快角色的速度。

  • 如果在角色攻擊時關閉 則角色會一直保持攻擊時的速度。

速度倍率 #

設定角色在攻擊時的速度倍率。

  • 範圍: 1 ~ 5

自定義元素 #

使角色造成的所有傷害為指定元素。

  • 一共有三種模式: 、以及。

  • 可以作用於生物。

  • 多人遊戲下僅限成為時對一定範圍內的生物有效。

隨機元素類型 #

隨機選擇造成傷害的元素類型。

  • 會覆蓋下面的。

  • 隨機選擇11種: 火 水 風 雷 草 冰 岩 冰封 免疫 楓丹小艇反凍結 未知

元素類型 #

指定造成傷害的元素類型。

  • 可指定12種: 無元素 火 水 風 雷 草 冰 岩 冰封 免疫 楓丹小艇反凍結 未知

元素佇列模式 #

自定義造成傷害的元素類型的順序。

  • 會覆蓋上面的。

添加 #

添加選擇的元素至列表最後。

清除 #

刪除清單所有項。

移除最後一個 #

刪除清單中最後一項。

無攻擊時自動重置佇列 (重置時間(毫秒)) #

經過設定的時間沒有角色造成傷害後 下一次攻擊將從第一項元素開始。

  • 範圍: 1000 ~ 600000

延遲切換元素 (延遲時間(毫秒)) #

一次切換元素后需要經過設定的時間才能進行下一次切換。

  • 範圍: 1 ~ 1000

自動弱點 #

使某些生物承受弓箭角色的攻擊時都在主要位置上。

  • 現版本 此功能無法生效。

刀刀暴擊 #

使每一次攻擊都能造成暴擊。

  • 不能對心海等負暴擊率的角色生效。

  • 多人遊戲下 僅自己有效。

武器附魔 #

使單手劍、雙手劍以及長柄武器角色的武器附魔指定元素。

  • 連線狀態下 武器附魔特效僅自己可見。

  • 有效果但沒有顏色的附魔: 草 岩 冰封 免疫 楓丹小艇反凍結 未知

隨機元素類型 #

隨機選擇的元素類型。

  • 會覆蓋下面的。

  • 隨機選擇11種: 火 水 風 雷 草 冰 岩 冰封 免疫 楓丹小艇反凍結 未知

元素類型 #

指定的元素類型。

  • 可指定12種: 無元素 火 水 風 雷 草 冰 岩 冰封 免疫 楓丹小艇反凍結 未知

滿元素爆發能量 #

  • 若有角色瞬間蓄力無效 請在 此Discord頻道 上提供反饋。

無敵模式 #

使角色不會受到來自魔物以及環境的任何傷害。

  • 不會累計嚴寒值以及一些與其類似的某些機制。

觸發條件 #

設定啟用的前置條件。

最小生命值 #

設定百分比超過上場角色生命值百分比時才可觸發。

  • 範圍: 0.1 ~ 100

攻擊閃避率 #

設定值觸發的百分比概率。

  • 範圍: 0 ~ 100

不死 #

免疫環境傷害，並閃避所有大於角色當前血量的數值傷害。

  • 會使累計的嚴寒值以及一些與其類似的某些機制無法通過篝火、傳送錨點等減少。

  • 在應對某些攻擊時可能不會即時生效。

無摔落 #

免疫摔落傷害。

無限體力 #

使角色的體力不再減少。

啟用 #

使衝刺、攀爬、游泳、飛行不再消耗體力。

  • 有可能造成無法衝刺、攀爬、飛行等 需要重啟遊戲以恢復。

阻止上傳模式(更安全) #

會攔截衝刺、攀爬、飛行等體力消耗動作的伺服器數據包。

  • 聯機時隊友會看到你的角色行為異常 如閃現、水中行走等。

自由移動 #

替換原有移動機制 使角色可以在世界中自由移動。

啟用 #

是否啟用。

  • 聯機時會看到其他玩家異常移動。

  • 其他玩家也許不能看到你在空中飛行 而只是在地上行走。

  • 預設按鍵: W - 向前移動 | S - 向後移動 | A - 向左移動 | D - 向右移動 | Space - 向上移動(相對角色) | LCtrl - 向下移動(相對角色)

允許在空中釋放技能 #

使角色啟用自由移動后可以一直保持如站立、墜落、游泳、飛行等動作狀態。

  • (使角色在空中可以保持站立狀態而不會進入墜落狀態。 )

  • 需要在站立狀態下啟用才可以在空中釋放技能。

速度 #

設定自由移動的速度。

  • 超過35可能會被伺服器拉回。

  • 範圍: 2 ~ 100

相對相機方向移動 #

使用 W S A D 或 控制時相對於相機方向 而不是角色朝向。

備用自由移動 #

按住 LCtrl 或 使用時不會使用上面的而是改用下面的。

變速 #

設定備用自由移動速度。

  • 範圍: 2 ~ 100

加速度模式(暫) #

給角色增加重力以模擬真實的行走方式 解決部分行走穿模問題。

  • 可以解決在不啟用和時不能向角色其他方向移動的問題。

自由飛行模式 #

不移除碰撞體。

  • 在速度過慢時 你可能無法穿過部分地形。
