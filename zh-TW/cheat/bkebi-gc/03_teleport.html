如何快速跑圖。

寶箱傳送 #

檢查周圍選定寶箱或見聞的有效點位並傳送。

就近傳送 #

使用指定按鍵傳送到最近的選定有效點位。

  • 若周圍存在有效點位 則會在 信息視窗 顯示有效點位的距離。

  • 在沒有有效點時 使用按鍵將不會進行傳送。

顯示資訊 #

在 更多界面 - 信息視窗 顯示關於最近寶箱的簡易資訊。

  • 現版本 此功能僅會保持一直顯示。

過濾器 #

設定的生效類型。

過濾類型 #

設定需要檢測的寶箱或見聞。

  • 可選擇: 寶箱 摩拉箱 調查 書頁 唱片 任務交互 地脈之花 未知

品質過濾器 #

設定需要檢測的寶箱品質。

  • 需要在中勾選寶箱才能啟用勾選。

  • 可選擇: 普通 精緻 珍貴 華麗 奇饋

狀態過濾器 #

設定需要檢測的寶箱狀態。

  • 需要在中勾選寶箱才能啟用勾選。

  • 可選擇: 鎖定 岩封 冰封 在荊棘中 陷阱

道具 #

顯示周圍所有選定寶箱或見聞的有效點位。

  • 可以點擊『Teleport』傳送到指定的點位。

  • 若大世界存在的寶箱沒有被Bkebi-GC標記為寶箱 則在 此Discord頻道 上提供反饋。

自定義傳送 Pro #

創建或載入自定義傳送節點以快速收集等。

啟用 #

是否啟用。

  • 使用旅行者傳送時有可能會被傳送至最近的錨點。

自動傳送 #

根據節點順序從所選節點開始依次傳送。

智慧傳送 #

按距離由近及遠傳送 而不是節點順序。

跳過空月祝福 #

跳過凌晨四點的空月祝福界面。

智慧降低傳送延遲 #

如果距離下一個節點不足50米 則會通過設定的倍率來減少延遲以提高效率。

  • 例: 為20 設定此項為0.75 距離不足50米時 此次延遲為20*0.75=15秒。

  • 啟用與以使用此項。

  • 範圍: 0.25 ~ 1

隨機延遲 #

使用隨機延遲自動傳送。

  • 此選項將覆蓋。

最小延遲 (秒) #

設定自動傳送的最小延遲。

  • 範圍: 5 ~ 359

最大延遲 (秒) #

設定自動傳送的最大延遲。

  • 範圍: 6 ~ 360

自動傳送延遲(秒) #

設定自動傳送的延遲。

  • 範圍: 5 ~ 360

傳送下一個 #

使用按鍵快速傳送到下一個傳送節點。

傳送上一個 #

使用按鍵快速傳送到上一個傳送節點。

快速將當前位置添加到當前數據 #

使用按鍵快速將當前角色的位置添加到當前選擇的傳送檔中。

  • 需要當前地圖與的相同。

  • 需要處於勾選狀態 才能夠被添加。

  • 距離現存節點的位置大於1米 才能夠被添加。

自動傳送結束後執行命令行 #

在自動傳送的所有點位傳送完成後執行一次命令行。

命令行 #

設定需要執行的命令。

  • 例: shutdown -s -t 60(一分鐘后關閉計算機)。

檔案清單 #

管理現有所有傳送檔。

  • 所管理的檔在『./tp_files』目錄。

  • 自Bkebi-1.1.0起 舊版Json傳送檔需要轉換才可以使用 詳見 JSON轉BPFC 。

  • 可載入第三方傳送檔 有關獲取傳送檔以及更多資訊請移步至 JSON傳送 。

  • 自定義傳送相關工具: Teleports-editor

名稱 #

設定將要創建或修改的傳送文件的昵稱。

描述 #

設定將要創建或修改的傳送文件的描述。

建立新檔 #

創建一個包含在上面設定的和的當前地圖傳送檔。

  • 輸出到『./tp_files』目錄。

重新載入檔 #

重新載入所有傳送檔。

  • 讀取『./tp_files』目錄。

  • 進入介面 會自動載入檔一次。

打開資料夾 #

使用資源管理器打開存放傳送檔的『./tp_files』目錄。

合併選定檔 #

創建一個包含在上面設定的和並含有選定傳送檔中的所有節點的傳送檔。

  • 若地圖不相同 可能無法正常合併。

  • 輸出到『./tp_files』目錄。

保存更改 #

更改為上面設定的和。

當前傳送檔 #

目前正在進行的傳送檔。

當前傳送文件地圖 #

目前正在進行的傳送檔所對應的地圖。

  • 地圖: 3 - 提瓦特 | 5 - 淵下宮 | 6 - 層岩巨淵: 地下礦區

搜索 #

使用關鍵詞篩選傳送檔。

選擇 #

載入此項傳送檔以從此項的節點開始傳送。

重置傳送記錄 #

重置此項傳送檔裡的所有節點『已傳送的』狀態。

刪除檔案 #

刪除此項傳送檔。

清除所選記錄 #

重置所選傳送檔裡的所有節點『已傳送的』狀態。

刪除選定的檔案 #

刪除選取選取檔案。

傳送節點 #

管理的所有傳送節點。

  • X Y Z 即為節點數據。

  • 若傳送到某個傳送節點 則對應的『已傳送的』狀態會標記為『是』。

傳送 #

傳送到此節點。

刪除 #

刪除此節點。

添加當前位置座標 #

添加一項當前位置的節點。

  • 需要當前地圖與的相同。

  • 需要處於勾選狀態 才能夠被添加。

  • 距離現存節點的位置大於1米 才能夠被添加。

刪除選定位置座標 #

刪除所選取的節點。

地圖傳送 #

在地圖上使用『滑鼠左擊』一鍵傳送。

啟用 #

是否啟用。

使用原座標 #

在傳送建築物或錨點時使用系統預設傳送位置 而不是指定的絕對位置。

指定高度 #

在無法獲取到地面高度時 會將角色傳送至此高度。

傳送鍵 #

設定使用的前置按鍵。

  • 設定的快捷鍵不為 None 時 需要在按住時使用滑鼠左擊傳送。

  • 設定的快速鍵為 『None』 時 僅使用『滑鼠左擊』即可傳送。

  • 若需要切換生效 則設定此處為 None 並在 熱鍵 - 地圖傳送 設定按鍵 。

神瞳傳送 #

檢查周圍神瞳的有效點位並傳送。

就近傳送 #

使用指定按鍵傳送到最近的有效點位。

  • 若周圍存在有效點位 則會在 信息視窗 顯示有效點位的距離。

  • 在沒有有效點位時 使用按鍵將不會進行傳送。

神瞳 清單 #

顯示周圍所有神瞳的有效點位。

  • 可以點擊『Teleport』傳送到指定的點位。

任務點傳送 #

按下快捷鍵快速傳送到任務追蹤目標。

任務點傳送 #

是否啟用。

傳送快捷鍵 #

  • 使用指定按鍵傳送到已追踪的任務目標。

  • 設定的快捷鍵為 None 時 將無法生效。
